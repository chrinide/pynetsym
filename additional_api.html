class pynetsym.termination.ICondition¶

Condition interface.

Conditions are object that provide a check method that receives a graph and answers whether the simulation has to stop. After the condition evaluated to True a motive attribute indicates the termination reason. It is intended to be meaningful for the user and not for the machine.
